# Smart-Placement-Tracking-System
The Smart Placement Tracking System is a MERN stack web application developed to simplify and automate the placement management process in colleges.It enables students to create profiles, apply for placement drives, and track their application status, while placement officers can manage companies, job postings, eligibility criteria, interview rounds, and placement records through a centralized dashboard.

Tech Stack

MongoDB – Database for storing student, company, and placement data
Express.js – Backend framework for handling APIs and server-side logic
React.js – Frontend library for building the user interface
Node.js – Runtime environment for executing backend services

Features

Student profile management
Company and placement drive management
Eligibility checking system
Application tracking and status updates
Admin and student dashboards
Secure authentication and authorization
Placement reports and analytics
